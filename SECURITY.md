# Security Policy

## Reporting

Please report security issues privately to the repository owner instead of opening a public issue.

Include:

- Affected version or commit.
- Reproduction steps.
- Expected impact.
- Suggested mitigation if known.

## Scope

Relevant security areas include:

- Unsafe file path handling.
- Archive extraction traversal or overwrite issues.
- Host resolver script execution boundaries.
- Download URL handling and unexpected redirects.
- Persistence file corruption or tampering.

FFDOWNLOADER is a local desktop application. It should not be used to download or extract files from untrusted sources without normal Windows security precautions.
